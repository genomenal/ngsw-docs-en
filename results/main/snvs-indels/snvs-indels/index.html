SNVs/Indels

The sections "Somatic SNVs/Indels" and "Germline SNVs/Indels" contain the results of discovery of somatic and germline, respectively, single-nucleotide variants (SNVs) and short insertions/deletions (Indels), in the sample. Here are the variants after filtering with GATK Best Practices or all discovered variants if the sample file was uploaded in VCF, TXT or TSV format or if the Allow only variants passing all filters setting is disabled.

Somatic mutations occur after conception and are not passed on to children. They can cause cancer or other diseases. Somatic mutations discovery is carried out only for tumor tissue samples and, therefore, the results are presented only on the tumor tissue sample page.

Germline mutations can be discovered in any cells of the body and are passed on from parents to offspring. Accordingly, germline mutations discovery is carried out both for tumor tissue samples (in the case of single tumor tissue samples without control) and for non-tumor tissue samples. In the case of a set of tumor/normal samples, the results of the germline mutations discovery in the normal sample are presented on the pages of both samples of the set.

The detected and annotated variants are presented in both sections as:

  1. SNV Viewer is a embedded service for viewing and analyzing variants. You can read more about it in the corresponding section. Click on to open SNV Viewer.
  2. Text file in CSV (Comma-Separated Values) format. To download the file, click on .
  3. File in VCF 4.2 (Variant Call Format) format. To download the file, click on .
  4. File in Google Spreadsheet (only a limited number of variants can be shown). To open the file, click on .
  5. Integrative Genomics Viewer (IGV) is an embedded module for visualization of variants on the genome. To open the module, click on .
info

If you would like to add the track of annotated variants discovered in the sample analyzed by Genomenal to IGV desktop application, you can load it via URL. To do this, do the following:

  1. Right-click on and select the "Copy link address" option.
  2. Load the track via URL to the IGV desktop application as described here.
  3. Open the "Workflow details" tab on the sample page and expand the details of the "SNVs/Indels annotation" stage, and then the details of the "Annotate SNVs/Indels" task. Right-click on the link to the annotation index file "Download Filtered variants VCF_TBI" for a sample uploaded in FASTQ format*, or "Download All variants VCF_TBI" for a sample uploaded in VCF format, and select the "Copy link address" option.
  4. Add the index file URL in the corresponding field in IGV.
  5. Click "OK". Done! The track of annotated variants discovered in the sample has been added to IGV.

*If the parameter allowing only variants passing all filters has been switched off for the sample, then copy the link to all variants' annotation index file "Download All variants VCF_TBI".
