Alignments

The "Alignments" section contains the sample reads mapped to reference genome. Subsequently, during the analysis, it is these reads that are used to call variants.

The results of alignment of sample reads are presented as:

  1. File with alignments in BAM (Binary Alignment Map) format. To download the file, click on .

  2. Index file for alignment in BAI format. The file is required for some tools. To download the file, click on .

  3. Integrative Genomics Viewer (IGV) is an embedded module for visualization of variations on the genome. To open the module, click on .

If you would like to add an alignment track of the sample analyzed by Genomenal to IGV desktop application, you can load it via URL. To do this, do the following:

  1. Right-click on and select the "Copy link address" option.
  2. Load the track via URL to the IGV desktop application as described here.
  3. Right-click on the index file for alignment and select the "Copy link address" option.
  4. Add the index file URL in the corresponding field in IGV.
  5. Click "OK". Done! Sample alignment track has been added to IGV.
