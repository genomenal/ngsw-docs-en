Variant Viewer

Overview#

Variant Viewer is a built-in service for viewing and analyzing variants. Located on the main page of the sample ("Main" tab).

On the tumor sample page, you will find two Variant Viewers: one with somatic and one with germline mutations discovered in this sample:

The normal sample page presents a Variant Viewer with germline mutations only:

The main page#

After clicking on the Variant Viewer button, you will see the main page of the service:

The main object of the Variant Viewer is a table with variants - each variant on a new line:

Description of table columns#

By default, the table has the following columns (the set and order of the columns can be changed):

  1. LOCUS is the common name of the gene in which the variant is located.
  2. ORIGIN is a mutation type (somatic or germline).
info

"Origin" column is hidden in Variant Viewer with germline mutations and in Variant Viewer with somatic mutations if "Origin Somatic" filter is on.

  1. POSITION is the coordinates of the variant in the genome (chromosome + start position).
  2. HGVSP is an amino acid substitution using the HGVS notation (‚Äúp.‚Äù prefix (protein) + reference amino acid + amino acid position in protein + new amino acid resulting from the substitution). When hovering over the value of the HGVSP column for a variant, you can see the transcript which the substitution was predicted for and HGVSp with amino acids in the three letter code:

  1. ALTERATION is a nucleotide substitution (reference allele > alternative allele(s)).
  2. HGVSC is a nucleotide substitution using the HGVS notation (genomic position of the substituted nucleotide + reference allele > alternative allele). Starts with a prefix: ‚Äúc.‚Äù (coding; for a substitution in the coding sequence), ‚Äún.‚Äù (non-coding; for a substitution in the non-coding sequence). When hovering over the value of the HGVSP column for a variant, you can see the transcript which the substitution was predicted for:

  1. EXON is the exon number in the transcript which is affected by the variant.
  2. DEPTH (ALT/REF) is a sequencing depth; the total number of reads of the sequence overlapping the variant position for this sample. ALT is the alternative allele reads count, REF - the reference allele reads count.
  3. GT is a genotype; allelic values (0 - reference allele, 1 - first alternative allele, 2 - second alternative allele etc.), divided by ‚Äù/‚Äù (for unphased genotype) or ‚Äú|‚Äù (for phased genotype).
  4. CONSEQUENCE is the effect of the variant on genes.
