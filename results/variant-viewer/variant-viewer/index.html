Variant Viewer

Overview#

Variant Viewer is a built-in service for viewing and analyzing variants. Located on the main page of the sample ("Main" tab).

On the tumor sample page, you will find two Variant Viewers: one with somatic and one with germline mutations discovered in this sample:

The normal sample page presents a Variant Viewer with germline mutations only:

The main page#

After clicking on the Variant Viewer button, you will see the main page of the service:

The main object of the Variant Viewer is a table with variants - each variant on a new line:

Variant information#

Besides of the information presented in the table (see Description of table columns), for each variant there are several built-in modules with additional information:

  1. Variant detailed information panel. Click on the variant row to see a panel:

You can read more about the panel and its tabs in the corresponding section.

  1. Variant detailed information page - here you can add the variant interpretation, include the variant to the report, and much more. Click on the highlighted button in the variant line to open the page:

  1. ACMG Classification - is a page with the criteria developed by ACMG for the variant interpretation. Click on the highlighted button in the variant line to open the page:

  1. Variant visualization module (IGV). Click on the IGV button in the variant line to open the built-in module:

Pinning variants#

The variants you want can be pinned. Click on the paperclip in the variant line to pin it:

The pinned variant will appear at the top of the table (the row highlighted in blue):

info

Variant pinning remains when leaving the Variant Viewer page.

Sorting and filtering variants#

You can sort the table in ascending or descending order of the values of the LOCUS, ORIGIN, POSITION, EXON, DEPTH, CONSEQUENCE, IMPACT, GNOMAD AF, ACMG columns. Click on the name of the corresponding column to sort by it (one click to sort in ascending order, two clicks - in descending order).

Filtering of variants in Variant Viewer is described in the Basic filtering and Advanced filtering sections.

Export of variants#

Variant Viewer allows you to export variants in CSV format:

  • Export of all variants: click on and select the "All to CSV" option:

  • Export of only filtered variants: filter the variants you want by Basic or Advanced filtering, then click on and select the "All to CSV" option:

  • Export of only pinned variants: pin the variants you want, then click on and select the "Pinned only to CSV" option:

Description of table columns#

By default, the table has the following columns (the set and order of the columns can be changed):

  1. LOCUS is the common name of the gene in which the variant is located.
  2. ORIGIN is a mutation type (somatic or germline).
info

"Origin" column is hidden in Variant Viewer with germline mutations and in Variant Viewer with somatic mutations if "Origin Somatic" filter is on.

  1. POSITION is the coordinates of the variant in the genome (chromosome + start position).
  2. HGVSP is an amino acid substitution using the HGVS notation (“p.” prefix (protein) + reference amino acid + amino acid position in protein + new amino acid resulting from the substitution). When hovering over the value of the HGVSP column for a variant, you can see the transcript which the substitution was predicted for and HGVSp with amino acids in the three letter code:

  1. ALTERATION is a nucleotide substitution (reference allele > alternative allele(s)).
  2. HGVSC is a nucleotide substitution using the HGVS notation (genomic position of the substituted nucleotide + reference allele > alternative allele). Starts with a prefix: “c.” (coding; for a substitution in the coding sequence), “n.” (non-coding; for a substitution in the non-coding sequence). When hovering over the value of the HGVSP column for a variant, you can see the transcript which the substitution was predicted for:

  1. EXON is the exon number in the transcript which is affected by the variant.
  2. DEPTH (ALT/REF) is a sequencing depth; the total number of reads of the sequence overlapping the variant position for this sample. ALT is the alternative allele reads count, REF - the reference allele reads count.
  3. GT is a genotype; allelic values (0 - reference allele, 1 - first alternative allele, 2 - second alternative allele etc.), divided by ”/” (for unphased genotype) or “|” (for phased genotype).
  4. CONSEQUENCE is the effect of the variant on genes.

Impacts of variants in the annotation#

ImpactDescription
HIGHThe variant has high (disruptive) impact in the protein, probably causing protein truncation, loss of function or triggering nonsense mediated decay
MODERATEThe variant might change protein effectiveness
LOWThe variant is mostly harmless or unlikely to change protein behavior
MODIFIERUsually variants affecting non-coding genes, where predictions are difficult or there is no evidence of impact
  1. EXTERNAL LINKS are links to pages with variant information in dbSNP, ClinVar databases.
  2. GNOMAD AF is the allele frequency according to the gnomAD database.
  3. ACMG is the interpretation of the variant by ACMG recommendations. If you have not interpreted the variant, then the column value will be “N/A” (No interpretation).
