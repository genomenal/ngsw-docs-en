# Quick Start for Prenatal Genetic Testing

If you would like to perform prenatal genetic testing with Genomenal, then this section is for you.

### 1. Create Patient {#create-patient}

<Login />

import PatientInfo from '/docs/imported_components/_patient_def.mdx';

To create a patient entry, click
on <img src={require('/img/eng/add_patient_btn.png').default} width="150"/> on the "All patients" page:

import PatientPage from '/docs/imported_components/_patient_page.mdx';

<PatientPage />

You will see the patient page:

<p align="center">
<img src={require('/img/eng/3a-patient-page.png').default} width="900"/>
</p>

### 2. Upload Samples {#upload-samples}

Select sample file(s) to analyze. Commonly, the sample is a paired-end sequencing data in FASTQ format.

<UploadFiles />

import FillCard from '/docs/imported_components/_fill_out_patient_card.mdx';

<p align="center">

</p>

### 3. Start Analysis {#start-analysis}

Click on <img src={require('/img/eng/start_analysis_btn.png').default} width="200"/> to start sample analysis.

import DangerUploading from '/docs/imported_components/_danger_while_uploading.mdx';

<DangerUploading />

You can see that the sample analysis is complete when all stage statuses are green or orange:

<p align="center">
<img src={require('/img/eng/7a-stage-statuses.png').default} width="150"/>
</p>

### 4. Analysis Results {#analysis-results}

When the analysis is complete, you can examine the results. Open the sample page by clicking on sample row:

<p align="center">
<img src={require('/img/eng/7b-sample-row.png').default} width="700"/>
</p>

Then click on the **CNV Report** tab and explore the report.

<p align="center">
<img src={require('/img/eng/33-cnv-report-chr21.png').default} width="800"/>
</p>

import SampleCNVReport from '/docs/imported_components/_sample_cnv_report.mdx';

<SampleCNVReport />

Report can be exported to PDF.

In addition, you can view the analysis results of all patient samples. You can find information on
how to do this [here](/results/cnv-report/patient-cnv-report).

Further, [here](/sample-examples/normal-no-cnvs) you can find examples of prenatal genetic testing
results for samples of different quality.
