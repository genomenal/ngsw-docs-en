Quality Control Settings

Quality control is an essential first step in sequencing data analysis. It includes checking quality of sample reads and, if some of the quality metrics were not met, cleaning the data up (trimming and/or filtering). Reads quality is checked using the Falco tool, which runs on average three times faster than the equivalent FastQC tool1. Reads are trimmed using the Cutadapt tool, which finds and removes adapter sequences, primers, poly(A) tails, and other types of unwanted sequence from reads2. Reads are filtered using the FASTX-Toolkit Quality Filter tool from FASTX-Toolkit, which filters sequences based on quality.

Disable quality control#

Number of readsMin Total Sequences is the minimal meaningful number of reads per file. If the sample file contains fewer reads before or after cleaning, then automatic sample processing stops.10,000
Reads length distributionShort Sequence Bound is the maximal length of reads per file. Reads shorter than this value are discarded in processing.20
Max Short Sequences Percent is the maximal percent of short reads per file. If this threshold is exceeded, the file is considered faulty by length distribution.25
Tiles reads qualityMax Tile Quality Deviation is the maximal acceptable tile quality deviation per file. Tiles that exceed mean quality across all tiles by this threshold are considered faulty.7
Max Tile Bad Quality Cells is the maximal acceptable number of bad quality cells in a tile per file. If this threshold is exceeded, the cleaning by tiles is activated.10
Per base reads qualityBase Start Count is the number of nucleotides that represent the start of a read when checking quality distribution across all reads.3
Base End Count is the number of nucleotides that represent the end of a read when checking quality distribution across all reads.3
Min Base Percentile Quality is the minimal acceptable quality of a position in a read. When evaluating, the first (in the amount equal to Base Start Count), last (in the amount equal to Base End Count) and middle bases are considered separately. If this threshold is exceeded, the trimming/filtering of reads is performed.20
Overrepresented sequencesMax Overrepresented Sequences Percent is the maximal percent of overrepresented sequences per file. If this threshold is exceeded, the file is considered faulty by overrepresented sequences.1
Adapter contaminationMax Adapter Contaminated Percent is the maximal percent of reads contaminated with adapter sequences per file. If this threshold is exceeded, adapter clipping is activated.1
Base N contentMax N Content Percent is the maximal percent of unknown nucleotides (N) in any position among all nucleotides per file. If this threshold is exceeded, the file is considered faulty by base N content.20
GC contentMin Derivative Threshold is the minimal threshold for detecting the local extremes of GC per file. If secondary extremes are detected, the file is considered faulty by GC content.0.085
Max Wave Sloping Threshold is the maximal threshold for the slope of the local extreme of GC. Exceeding this threshold indicates GC-content disturbances, and in this case the file is considered faulty by GC content.0.035
Min Wave Length is the minimal acceptable length of GC-content disturbances (distance between two local extremes of GC). Exceeding this threshold indicates GC-content disturbances, and in this case the file is considered faulty by GC content.4
Base sequence content (AT/GC ratio)Max Cutoff at End is the number of nucleotides from 3'-end that are skipped when evaluating GC-content.3
Max Cutoff at Start is the number of nucleotides from 5'-end that are skipped when evaluating GC-content.15
Cutoff Difference Percent is the difference threshold between base pairs (in percent). If the difference between A and T, or G and C exceeds this threshold, then the end nucleotides are skipped.1
Mean Difference Threshold is the threshold of the mean of difference between the base pairs (in percent). If the mean of difference between A and T, or G and C exceeds this threshold, then the file is considered faulty by base sequence content.1
Max Difference Threshold is the threshold of the maximum difference between base pairs (in percent). If in any position the difference from all reads maximum between A and T, or G and C exceeds this threshold, then the file is considered faulty by base sequence content.20

Change metric thresholds#

To change the metric threshold value, click on the current value and edit. To save changes, press Enter or click outside edit value container.
