Alignment Settings

Alignment of sequence reads onto a reference genome is the most fundamental step for any NGS data analysis. GRCh38.p14 version of the reference human genome is used for alignment. By default, alignment is performed using the BWA-MEM2 (BWA2 Burrows-Wheeler Aligner) tool, which is the next and faster version of the BWA-MEM algorithm, which is based on backward search with Burrowsâ€“Wheeler Transform (BWT) and efficiently and quickly aligns short sequencing reads against the human genome, allowing mismatches and gaps1. You can find out how to change the alignment tool in the corresponding section.

Disable alignment#

info

If alignment is disabled, further analysis (SNVs/Indels, structural variations, and copy number variations discovery) is not possible.

If you want to disable alignment, turn off the corresponding toggle:

Disable re-pair#

When aligning paired-end sequence (in this case, both ends of a DNA fragment are sequenced towards each other on opposite strands), it is essential that each read from one file contains a mate in the second. Therefore, before alignment, the pairing of reads is ensured and restored. Pairing (number and order of reads) may not match in input paired-end sequencing files. In addition, pairing can be disturbed during cleaning the data, then it is necessary to remove reads that have lost a mate. If the number of reads in the files does not match, then the re-pair stage cannot be avoided, it will proceed even if it is disabled. However, if you are confident in your data quality and want to speed up the analysis process, then you can disable this stage by turning off the toggle:

Change alignment tool#

In addition to the default BWA-MEM2 (BWA2 Burrows-Wheeler Aligner), the following alignment tools are available for selection:

  • BWA-MEM (BWA Burrows-Wheeler Aligner), an earlier version of the BWA-MEM algorithm2.
  • Bowtie23
  • HISAT24
  • STAR5

To change the alignment tool, click on the value of the "Alignment tools" field and select the aligner you need:

Use singleton reads#

If a read in one file has no mate in paired file, it is called a singleton read. Such situation can occur if paired-end and single-read sequencing results are mixed in one file, or if paired read was lost during data pre-processing (for example, during cleanup). If the "Use singleton reads" option is disabled, then singleton reads are discarded from further analysis at the re-pair stage. If you want to use singleton reads in alignment, turn on the corresponding toggle:

note

This option is only applicable to BWA or BWA2 aligners.

Sequencing type detection#

If you know exactly what sequencing type was used to obtain your data, you can choose the appropriate type to speed up analysis. This can be whole genome sequencing (WGS) or targeted sequencing (exome or panel). To select the sequencing type, click on the value of the "Sequencing Type" field and select the type you need:

To analyze your sample with a targeted panel, define a Capture Kit for more accurate and faster analysis. Capture Kit is a bed file provided for your exome capture kit and represents targeted features that you are expecting to see in the sequencing. You can choose a bed file appropriate for your exome capture kit from the built-in and uploaded kits:

You can upload your Capture Kit on the "Capture kits" page from the "Settings" section. Usually appropriate Capture Kit can be obtained from kit producer website.

Alignment metrics#

MetricMetric thresholdThreshold default value
Mapped readsMin mapped reads percent is the minimum acceptable percent of mapped reads per alignment file.85
Multiple alignmentsMax multimaps percent is the maximum acceptable percent of multiple alignments of same read per genome in alignment file.15
Forward/reverse balanceForward/reverse max difference is the maximum acceptable difference between forward and reverse strand reads per alignment file.10
Paired mapped readsMin paired mapped percent is the minimum acceptable percent of paired mapped reads per alignment file.80
Paired properly mapped readsMin paired properly mapped reads percent is the minimum acceptable percent of paired properly mapped reads per alignment file.75
Coverage per nucleotideMin coverage per nucleotide is the minimum acceptable coverage per nucleotide.0.1

If the metric value in the sample alignment file does not meet the set threshold, then the alignment is considered faulty by this metric, which is noted in the alignment quality warning (the alignment completes with an error, the message of which contains the number of failed metrics).

Change metric thresholds#

To change the metric threshold value, click on the current value and edit. To save changes, press Enter or click outside edit value container.
